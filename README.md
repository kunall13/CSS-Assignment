# Laundry Service Web App

## 📌 Project Description
This is a simple web application UI for a laundry service.  
It includes a navigation bar and a hero section designed using HTML and CSS.  
The layout is responsive and adapts to different screen sizes.

## Features
- Responsive Navbar (Home, Services, About Us, Contact Us)
- Hero Section with call-to-action
- Modern UI design
- Mobile-friendly layout

## How to Run
1. Download or clone the project
2. Open the folder
3. Double-click on index.html

OR

Right-click → Open with browser

## Technologies Used
- HTML5
- CSS3 (Flexbox, Media Queries)
